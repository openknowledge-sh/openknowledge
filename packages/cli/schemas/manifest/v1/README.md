# Open Knowledge portable bundle manifest schema v1

`bundle.schema.json` describes the `openknowledge.json` discovery manifest
written by default HTML exports and consumed by `openknowledge connect`.

The website build publishes it at its declared identifier:

`https://openknowledge.sh/schemas/cli/manifest/v1/bundle.schema.json`

The manifest's `version` identifies this discovery/archive protocol. Its
concrete `spec` field independently identifies the Open Knowledge Format
revision required by the archived bundle.
