# Embedded OKF Specs

This directory stores pinned upstream copies of Open Knowledge Format (OKF)
specifications for offline CLI validation and scaffolding.

The embedded specifications are upstream material from the
GoogleCloudPlatform Knowledge Catalog repository, not original Open Knowledge
CLI work.

## License And Attribution

The upstream Knowledge Catalog repository is licensed under Apache-2.0. Keep a
copy of the upstream license in
`third_party/licenses/GoogleCloudPlatform-knowledge-catalog-Apache-2.0.txt`.

When adding or updating a spec copy:

1. Record the upstream URL, version, and observed modification timestamp here.
2. Preserve upstream copyright, patent, trademark, and attribution notices.
3. If the local copy is edited, add a prominent note describing the local
   change.
4. If upstream adds a NOTICE file relevant to the spec, copy the relevant
   attribution into `THIRD_PARTY_NOTICES.md`.

Open Knowledge CLI is unofficial tooling for the upstream specification and is
not an official Google product.
