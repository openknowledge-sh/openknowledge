From Markdown to shared context

Create your project wiki

Launch an agent to inspect the current repository, create the wiki, validate it, and install project integration.

openknowledge setup Wiki --from .
openknowledge view Wiki

Build from another source

Create, validate, and integrate a wiki grounded in a repository, folder, or website.

openknowledge setup Wiki --from https://github.com/openknowledge-sh/openknowledge --type understanding
openknowledge setup Wiki --from https://openknowledge.sh/wiki/ --type understanding --depth 2

Connect and retrieve

Give the bundle a stable name, then search it or browse it locally.

openknowledge connect ./project-memory --as personal
openknowledge search personal "validation workflow"
openknowledge view personal

Validate before use

Catch structure and link issues before agents or publishing depend on it.

openknowledge validate ./project-memory
openknowledge list --depth 2 ./project-memory
openknowledge ast ./project-memory

Publish or integrate

Ship a static site or export portable machine-readable views.

openknowledge export html --out ./project-site ./project-memory
openknowledge export json ./project-memory
openknowledge export tar --out ./project-memory.tar.gz ./project-memory
