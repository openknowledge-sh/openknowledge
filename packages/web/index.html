Open Knowledge

Create a local wiki that agents and humans can both navigate.

new

Scaffolds a local wiki with agent rules, setup handoff, indexes, logs, and a pinned spec. 

openknowledge new ./project-memory

validate

Checks the bundle against the selected Open Knowledge Format spec and reports structural issues. 

openknowledge validate ./project-memory

list

Prints the wiki tree and marks missing or invalid files inline, so the structure is easy to inspect. 

openknowledge list ./project-memory
Unofficial tooling for the Open Knowledge Format specification defined in the GoogleCloudPlatform knowledge catalog.
