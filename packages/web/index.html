Create a local LLM wiki with your coding agent

Open Knowledge helps you create and maintain local LLM wikis that are both human-readable and agent-readable. It keeps your knowledge base portable and consistent using the Open Knowledge Format v0.1 spec.

Paste this into your agent

Set up an Open Knowledge agentic wiki for this workspace. Install the Open Knowledge CLI if needed, run openknowledge setup, ask me the setup questions, customize the wiki for this workspace, validate it, and show me how to open it.

Need CLI shortcuts, manual installs, or advanced setup options? See the GitHub docs.

What Open Knowledge gives you

A wiki can live inside a project repo or stand alone as a private knowledge base. It stays readable in Git and easy for both humans and coding agents to navigate.

  • Turn a project, research folder, or private knowledge dump into a wiki that humans can read and agents can use.
  • Guide the first setup through an agent interview, so the wiki starts with the right purpose, structure, and maintenance habits.
  • Keep the result portable with Markdown files, local viewing, and validation against embedded OKF v0.1 rules.
